По статье: (разница² → усреднение по каналам) → сглаживание
У вас: (разница² → сглаживание 3D) → усреднение по каналам

1. Сглаживание 3D массива вместо 2D
Вы сглаживаете 3-канальный массив (H,W,3), а нужно сглаживать уже усреднённый 2D массив (H,W).

2. np.mean(comparison, axis=2) даёт СРЕДНЕЕ квадратов, а не квадрат разности усреднённых значений
Неправильно: mean[(R1-R2)², (G1-G2)², (B1-B2)²]
Правильно: [mean(R1,G1,B1) - mean(R2,G2,B2)]²
