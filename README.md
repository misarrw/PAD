# Инструкция к использованию PAD
1. В .env указать следующие переменные:
  - SAVE: путь к директории, куда будут сохраняться защищенные изображения
  - INPUT: путь к директории, где хранится датасет, который необходимо защитить
  - SAM: путь к модели сегментации
2. Скачать модель сегментации SAM. Это можно сделать командой `wget https://dl.fbaipublicfiles.com/segment_anything/sam_vit_l_0b3195.pth` или установить любую другую версию модели SAM.
3. Импортировать репозиторий, необходимый для работы с моделью сегментации: `git clone https://github.com/facebookresearch/segment-anything.git`.
На данном этапе могут возникнуть ошибки из-за несовпадения путей. В таком случае можно удалить пустую папку `segment_anything`, переназвать импортированный репозиторий `segment-anything` в `segment_anything`. Ошибки должны исчезнуть.
4. Запустить PAD коммандой `python run-PAD.py` в терминале.
